# 改行
## Pythonのprint関数は標準で改行コードを末尾に追加して文字列を出力する

print("Hello")

print(123)

Hello

123

## カッコの中に「, end=""」を追加することで、改行をなくすことができる

print("Hello", end="")

print(123)

Hello123

## random関数
0から1までの間で、ランダムな数値を返す

random.random() ：0から1までのランダムな数値を出力する


## randint関数
指定した引数の範囲で、ランダムな整数の値を返す

random.randint(a, b) ：aからbまでの間のランダムな整数を出力する
