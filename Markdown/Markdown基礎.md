# Lists - リスト

## Bullet List - 順序なしリスト
文頭に「*」「+」「-」のいずれかを入れると順序なしリストになる。要スペース
* リスト１
+ リスト２
- リスト３

## Ordered List - 番号付きリスト
文頭に「数字.」を入れると番号付きリストになる。要スペース
1. リストA
2. リストB
3. リストC 

## Horizontal rules - 水平線
下記は全て水平線として表示される。最低３つ以上は必要。
* * *
***
*****
- - - 
---------

## チェックボックス
順序なしリストの記号の後ろに[ ]を入れるとチェックボックスが生成される。

チェックが入った状態のボックスを生成する場合は[x]を入力する。前後に要スペース
- [ ] タスク1
- [x] タスク2
